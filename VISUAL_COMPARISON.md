## 🎨 Color Legend

### HTML Editor Colors
- 🟣 **Pink** (#f472b6) - HTML Tags (`<div>`, `<style>`)
- 🟡 **Yellow** (#fbbf24) - Attributes (`class`, `id`)
- 🟢 **Green** (#86efac) - String Values (`"text"`)
- ⚫ **Gray** (#6b7280) - Comments (`<!-- -->`)
- 🔵 **Blue** (#93c5fd) - Regular text/punctuation

### JSON Editor Colors
- 🟡 **Yellow** (#fbbf24) - Property Names (`"step"`)
- 🟢 **Green** (#86efac) - String Values (`"left"`)
- 🟠 **Orange** (#fb923c) - Numbers (`100`, `1`)
- 🟣 **Purple** (#c084fc) - Keywords (`true`, `false`)
- 🔵 **Blue** (#93c5fd) - Operators (`{`, `[`)
