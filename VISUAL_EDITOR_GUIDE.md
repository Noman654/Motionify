# 🎨 Visual Code Editor - Like VS Code!

## ✨ What You Get Now

Your Reel Composer now has a **professional visual code editor** with all the features you'd expect from VS Code!

### 🎯 Features Included

#### 1. **Syntax Highlighting** 🌈
Beautiful, color-coded syntax just like VS Code:

**HTML Editor:**
- 🟣 **Tags** - Pink/Magenta (`<div>`, `<style>`)
- 🟡 **Attributes** - Yellow (`class`, `id`, `style`)
- 🟢 **Values** - Green (`"text"`, `'value'`)
- 🔵 **Punctuation** - Blue (`<`, `>`, `=`)
- ⚫ **Comments** - Gray (`<!-- comment -->`)

**JSON Config Editor:**
- 🟣 **Keywords** - Purple (`true`, `false`, `null`)
- 🟡 **Properties** - Yellow (`"step"`, `"video"`)
- 🟢 **Strings** - Green (`"value"`)
- 🟠 **Numbers** - Orange (`100`, `0.5`)
- 🔵 **Operators** - Blue (`{`, `}`, `[`, `]`)

#### 2. **Auto-Indentation** 📐
- Press **Enter** → Automatically maintains current indentation
- Perfect alignment without manual spacing
- Clean, readable code structure

#### 3. **Tab Support** ⭐
- Press **Tab** → Adds 2-space indent
- Press **Shift+Tab** → Removes indent
- Just like your favorite code editor!

#### 4. **Format Button** 💜
- One-click HTML formatting
- Automatically organizes and indents your code
- Makes messy code beautiful instantly

#### 5. **Search Functionality** 🔍
- Quick search with case-insensitive matching
- Highlights and scrolls to matches
- Perfect for large HTML files

#### 6. **Line Counter** 📊
- Real-time line count display
- Know exactly how much code you have
- Located in the top-right toolbar

## 🚀 How to Use

### **Editing HTML**
1. Go to **HTML** tab
2. Start typing - see instant syntax highlighting
3. Press **Enter** - auto-indentation works
4. Press **Tab** - add indent
5. Click **Format** - organize everything

### **Editing Config (JSON)**
1. Go to **Layout** tab
2. Edit JSON with full syntax highlighting
3. Brackets and braces color-coded
4. Numbers, strings, booleans all highlighted

### **Keyboard Shortcuts**
| Action | Shortcut |
|--------|----------|
| Indent | Tab |
| Unindent | Shift+Tab |
| Auto-indent on new line | Enter |
| Search | Click Search button |

   - After pasting code → Click Format
