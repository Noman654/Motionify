# 🚀 Quick Reference Guide - Enhanced HTML Editor

## New Features at a Glance

### 🔍 Search Feature
**Location:** HTML Tab → Top Toolbar → "Search" button

**How to use:**
1. Click the "Search" button (magnifying glass icon)
2. Search bar appears below toolbar
3. Type what you're looking for (e.g., "div", "canvas", "style")
4. Press **Enter** or click **"Find"**
5. Editor auto-scrolls and highlights the match

**Tips:**
- Search is case-insensitive (finds both "DIV" and "div")
- Press Enter for quick searching without clicking
- Click "Search" button again to hide the search bar

---

### 💜 Format HTML Button
**Location:** HTML Tab → Top Toolbar → "Format" button

**How to use:**
1. Click the purple "Format" button
2. Your HTML is instantly formatted with proper indentation
3. All tags are organized with 2-space indentation
4. Line breaks added for better readability

**Before Format:**
```html
<div><style>body{background:#000;}</style><canvas></canvas></div>
```

**After Format:**
```html
<div>
  <style>
    body {
      background: #000;
    }
  </style>
  <canvas></canvas>
</div>
```

---

### 📊 Line Counter
**Location:** HTML Tab → Top Toolbar → Right side

Shows the total number of lines in your HTML code in real-time.
Useful for understanding code size and navigation.

## Pro Tips 💡

1. **Format First:** Always click "Format" when starting to edit pasted HTML
2. **Search Often:** Use search to quickly jump to specific elements
3. **Check Lines:** Keep an eye on line count to monitor code growth
4. **Save Often:** Click "Apply Changes" button at bottom after editing

## Troubleshooting

**Q: Search doesn't find anything**
- Make sure you typed the search term correctly
- Check if the content actually exists in your HTML
- Search is working if the textarea gets focus

**Q: Format button didn't work**
- Click "Apply Changes" after formatting
- Check if your HTML has valid structure
- If HTML is already formatted, you might not see changes

**Q: Text still looks small**
- The font size is 14px (was 12px before)
- If you need even larger, you can zoom your browser (Cmd/Ctrl +)
